# 작은 대화 프로젝트

## 프로젝트 컨셉

### 키워드

**사계절**, **감성 채팅**

### 전체 컨셉

UI 전반에 '사계절' 콘셉트를 적용했으며, 현재는 **봄(Spring)** 과 **겨울(Winter)** 테마를 구현해  
두 계절에 어울리는 색상, 아이콘, 배경, 애니메이션 등을 통해 각기 다른 감성을 시각적으로 표현하고 있습니다.

### 🌸 봄(Spring) 테마

> 봄은 벚꽃이 만개하는 계절로, 따뜻하고 부드러운 분홍색 계열을 메인 컬러로 선택했습니다.

- **메인 컬러**: 벚꽃색 `#ffb7c5`
- **배경**  
  → **라이트 모드**: 아침 햇살 아래 흩날리는 벚꽃 느낌  
  → **다크 모드**: 밤하늘에 은은히 퍼지는 벚꽃 이미지
- **애니메이션**: 벚꽃이 떨어지는 애니메이션

### ❄️ 겨울(Winter) 테마

> 겨울은 눈 내리는 차가운 계절의 분위기를 담아, 시린 느낌의 파랑색 계열을 메인 컬러로 선택했습니다.

- **메인 컬러**: 차가운 겨울 블루 `#0c436e`
- **배경**  
  → **라이트 모드**: 새벽에 눈이 조용히 내리는 듯한 이미지  
  → **다크 모드**: 어두운 겨울 밤, 눈이 내리는 이미지
- **애니메이션**: 눈송이가 떨어지는 애니메이션

### 폰트

- **Pretendard Variable**  
  기본 폰트로 사용되었습니다.
- **Jua**  
  서비스 로고 및 메인 타이틀에만 사용되며, 손글씨 느낌으로 감성적 분위기를 전달합니다.
