### UI 스타일링 가이드

- **Hover / Active 효과**
  - `opacity` 조절로 부드러운 색상 전환 표현
- **다크/라이트 모드 배경 처리 방식**
  - 다크 모드에서는 `background: transparent`로 배경 그라디언트를 강조
  - 라이트 모드에서는 `background-color`를 활용해 요소 간 구분 강조
- **텍스트 가독성**
  - 배경과 텍스트 컬러 간 대비가 부족할 경우를 대비해 `text-shadow` 적용
  - 예: `text-white`가 밝은 배경 위에 있을 경우 윤곽이 흐려지는 문제를 방지
